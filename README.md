# Спецпроект zakon.kz × Яндекс 360

Коммерческое предложение в виде одностраничного сайта.

Открыть локально: скачать папку и открыть `index.html` в браузере.

## Состав

- `index.html` — разметка всех секций
- `style.css` — стили, палитра снята с исходной презентации (#000C2F, синий, оранжевый)
- `script.js` — счётчики цифр, появление блоков при скролле, параллакс, переключение вариантов кампаний
- `img/` — 3D-графика и мокапы
